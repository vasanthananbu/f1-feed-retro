## 🌟 Overview

F1 Feed Retro is a modern web application that brings the excitement of Formula 1 to your screen with a nostalgic, retro aesthetic. It leverages the **OpenF1 API** for real-time race data and uses **Google Gemini** to provide intelligent commentary and insights.

## ✨ Features

- **Retro UI:** A beautiful, terminal-inspired interface that captures the classic era of racing telemetry.
- **Real-time Data:** Live tracking of race positions, lap times, and intervals using the OpenF1 API.
- **AI Commentary:** Dynamic, context-aware race commentary generated by Gemini.
- **Responsive Design:** Optimized for both desktop and mobile viewing.

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher)
- A Gemini API Key from [Google AI Studio](https://aistudio.google.com/)

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/vasanthananbu/f1-feed-retro.git
   cd f1-feed-retro
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Configure Environment Variables:**
   Create a `.env.local` file in the root directory and add your Gemini API key:
   ```env
   GEMINI_API_KEY=your_api_key_here
   ```

4. **Run the development server:**
   ```bash
   npm run dev
   ```

## 🛠️ Tech Stack

- **Frontend:** Vite, React, TypeScript
- **Styling:** CSS (Retro/Terminal Theme)
- **API:** OpenF1 API
- **AI:** Google Gemini API
